## Fluxo de inicialização da sessão

1. Electron inicia aplicação

2. Cliente solicita token

Electron
   │
   │ POST /session/token
   ▼
NestJS (Cloud Run)

3. Backend gera ephemeral token

NestJS
   │
   │ token
   ▼
Electron

4. Cliente abre sessão realtime

Electron
   │
   │ WebSocket
   ▼
Gemini Live API
